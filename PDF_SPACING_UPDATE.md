# PDF Spacing Update - Better Layout Match

## Problem
The PDF generated by the cloud function appeared more "squashed" compared to the print dialog version. The content was compressed with minimal padding and small fonts, making it look cramped.

## Comparison
- **Before:** Tight spacing, small fonts, compressed layout
- **After:** Better spacing, larger fonts, fills page properly (matches print dialog)

## Changes Made

### Page-Level Spacing
- **Page padding:** 8px → **16px** (doubled for better margins)

### Header Section
- **Header margin-bottom:** 4px → **8px**
- **Header padding-bottom:** 2px → **4px**
- **Title font-size:** 18px → **20px**
- **Title margin-bottom:** 2px → **4px**
- **Subtitle font-size:** 9px → **10px**

### Info Boxes (Business | Customer | Appliance)
- **Grid gap:** 4px → **8px** (more space between boxes)
- **Box padding:** 4px → **8px** (more internal padding)
- **Margin-bottom:** 4px → **8px**
- **Title font-size:** 10px → **11px**
- **Title margin-bottom:** 2px → **4px**
- **Title padding-bottom:** 2px → **3px**
- **Content font-size:** 9px → **10px**
- **Content div margin-top:** 1px → **2px**
- **Logo max-height:** 40px → **50px**
- **Logo margin-bottom:** 2px → **4px**

### Certificate Columns (Sweeping | Checks)
- **Grid gap:** 4px → **8px**
- **Column padding:** 4px → **8px**
- **Margin-bottom:** 4px → **8px**

### Section Headers
- **Font-size:** 10px → **11px**
- **Margin-top:** 4px → **6px**
- **Padding-top:** 3px → **4px**
- **Margin-bottom:** 2px → **4px**

### Sub-Headers
- **Font-size:** 9px → **10px**
- **Margin-bottom:** 1px → **2px**

### Fields
- **Font-size:** 9px → **10px**
- **Margin-top:** 2px → **3px**

### Condition Checks Grid
- **Grid gap:** 4px → **6px**
- **Margin-top:** 2px → **4px**
- **Column gap:** 2px → **3px**
- **Check item font-size:** 9px → **10px**

### Sign-Off Section
- **Section padding:** 4px → **8px**
- **Section margin-bottom:** 4px → **8px**
- **Row gap:** 4px → **8px**
- **Row margin-bottom:** 4px → **6px**
- **Row font-size:** 9px → **10px**

### Signatures
- **Grid gap:** 4px → **8px**
- **Padding-top:** 4px → **8px**
- **Label font-size:** 9px → **10px**
- **Label margin-bottom:** 2px → **4px**
- **Container padding:** 2px → **4px**
- **Container min-height:** 40px → **50px**
- **Image max-height:** 32px → **40px**

### Footer
- **Margin-top:** 4px → **8px**
- **Padding-top:** 3px → **6px**
- **Font-size:** 9px → **10px**

## Summary of Changes

### Font Sizes
All font sizes increased by 1px:
- 9px → 10px (body text, labels, values)
- 10px → 11px (section headers, box titles)
- 18px → 20px (main title)

### Spacing/Padding
All spacing roughly doubled:
- Small spacing: 2px → 3-4px
- Medium spacing: 4px → 6-8px
- Large spacing: 8px → 16px

### Visual Elements
Increased size for better visibility:
- Logo: 40px → 50px
- Signature containers: 40px → 50px
- Signature images: 32px → 40px

## Result

✅ PDF now has better spacing and fills the A4 landscape page properly
✅ Matches the print dialog version much more closely
✅ Text is easier to read with larger fonts
✅ More professional appearance with better breathing room
✅ Still fits on single A4 landscape page
